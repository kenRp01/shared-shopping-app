# 今回の依頼内容

## 依頼タイトル

リスト切り替えUIとデザイン改善

## 今の課題

- リスト名タブの下に同じリスト名が再表示されていて冗長
- 別のリストを追加する導線が足りない
- リストタブの順番を直感的に入れ替えられない
- 全体の見た目をもう少し可愛くしたい
- マイリストの設定ボタンが使いやすくない

## やりたいこと

- [x] リスト名タブの下にあるリスト名表示を削除する
- [x] 別のリストを作成する機能を追加する
- [x] ホバーで移動して買いたい商品のタブ順を入れ替えられるようにする
- [x] デザインをもう少し可愛くする
- [x] マイリストの設定ボタンを使いやすいUIに変える

## 変更内容の詳細

### 1. 見た目

- 色: 少し可愛く、親しみやすいトーンにしたい
- 文字サイズ: 情報の強弱がわかるように整理したい
- アイコン: 必要なら活用したい
- 余白: すっきり見えるように調整したい
- 非表示にしたい情報: リスト名タブ下の重複したリスト名

### 2. 操作

- 追加したい操作: 別リストの作成
- 消したい操作: 重複した見出し表示
- タップ時の動き: 設定を開きやすくしたい
- スワイプやドラッグの希望: タブを移動して順番を入れ替えたい

### 3. リスト / カテゴリー

- デフォルト名: マイリスト
- 並び順: ユーザーが入れ替えられるようにしたい
- 切り替え方法: タブで切り替えたい
- 共有 / 個人の扱い: 現状維持

### 4. モバイル優先の要望

- 片手操作しやすくしたい場所: リスト切り替え、設定ボタン
- 1画面内に収めたい情報: 主要な買い物項目とリスト切り替え
- スクロールを減らしたい場所: 上部のリスト周り

## 優先順位

1. リスト名の重複表示を削除する
2. 別リスト作成とタブ入れ替えを追加する
3. デザインと設定ボタンの使いやすさを改善する

## 完了条件

- [ ] リスト名タブ下の重複表示が消えている
- [ ] 別リストを追加できる
- [ ] タブの順番を入れ替えられる
- [ ] 見た目が今より可愛くなっている
- [ ] 設定ボタンが押しやすく分かりやすい
